# Podcast CSVデータ作成ガイドライン

このドキュメントは、SRTまたはVTTファイルから学習アプリ用CSVデータを作成するためのガイドラインです。

## 1\. 作成の基本方針

### 🎯 目的

\*\*「英語を英語の語順のまま理解する（チャンク・リーディング）」\*\*ための学習教材を作成します。

### ✅ 基本ルール

1.  **チャンクの粒度**:
      * SRT/VTTの細かい改行にとらわれず、**「ネイティブスピーカーが意味の塊として捉える長さ」**（文節、フレーズ単位）にまとめてください。
      * 短すぎず長すぎず、1行で文脈が掴める程度が理想です。
2.  **全文使用**:
      * 文章が長くても**省略せず全文を使用して**ください。
      * 音声データの**最初から最後まで**、すべての発話に対応するデータを作成してください。15分から20分相当になるはずです。
3.  **テキスト修正**:
      * 明らかに誤字・脱字がある場合や、文脈上不自然な単語は、正しい英語に修正してください。

### ⚠️ 区切り文字のルール（厳守）

CSVの構造を維持するため、以下の記号を使い分けてください。

  * **カラムの区切り**: **パイプ（ `|` ）**
  * **チャンク内の要素区切り**: **ハット（ `^` ）**

-----

## 2\. CSVフォーマット

### 📝 ヘッダー構成

| カラム名 | 説明 |
| :--- | :--- |
| **ID** | 通し番号 |
| **タイムスタンプ** | SRT/VTT由来の時間情報 |
| **英文** | 1行分の英文全体 |
| **日本語訳** | その英文の自然な日本語訳 |
| **チャンク1...n** | 文構造ごとに分解したデータブロック（数は可変） |

### 構造イメージ

```text
ID | タイムスタンプ | 英文 | 日本語訳 | チャンク1 | チャンク2 | チャンク3 ...
```

-----

## 3\. チャンク（構造分析データ）の書き方

各チャンク列には、以下の**4つの情報**をハット（`^`）で繋いで記述します。

### 記述フォーマット

`英語の塊`^`文法ラベル`^`タイプ`^`塊の訳`

### 各要素の詳細

| 要素 | 説明 | 例 |
| :--- | :--- | :--- |
| **英語の塊** | 意味の切れ目で区切った英語フレーズ | `I usually go` |
| **文法ラベル** | 文法的な役割 | `Subject + Verb (S+V)` |
| **タイプ** | 色分け用の分類（後述） | `core` |
| **塊の訳** | そのフレーズに対応する直訳 | `私はたいてい行きます` |

### 🎨 タイプ分類（色分け用）

| タイプ | 役割 | 色イメージ | 代表的な要素 |
| :--- | :--- | :--- | :--- |
| **`core`** | **文の骨格** | 🔴 赤色 | 主語(S)、動詞(V)、目的語(O)、補語(C) |
| **`modifier`** | **修飾語句** | 🟢 緑色 | 前置詞句、不定詞、副詞、形容詞句 |
| **`connector`** | **つなぎ言葉** | 🔵 青色 | 接続詞、関係代名詞、関係副詞 |

-----

## 4\. データ作成例

以下は実際のCSVデータの記述例です。

```csv
ID|タイムスタンプ|英文|日本語訳|チャンク1|チャンク2|チャンク3|チャンク4
1|00:00:05.714|A new Cold War has broken out between the U.S. and China over artificial intelligence.|人工知能を巡って、米国と中国の間で新たな冷戦が勃発しました。|A new Cold War has broken out^S+V^core^新たな冷戦が勃発しました|between the U.S. and China^Prep Phrase^modifier^米国と中国の間で|over artificial intelligence^Prep Phrase^modifier^人工知能を巡って|
2|00:00:12.718|The U.S. and China are locked in a competition, but there are some really striking similarities...|米国と中国は競争状態にありますが、冷戦時代に見られた技術競争と驚くべき類似点があります...|The U.S. and China are locked^S+V^core^米国と中国は膠着状態（競争中）です|in a competition^Prep Phrase^modifier^競争の中で|but^Conjunction^connector^しかし|there are some really striking similarities^There+V+S^core^実に驚くべき類似点がいくつかあります
```
