# 音声ファイル
- Distinction 2000の音声ファイルのうち、50kb以上のサイズのものを使用しています。
- サイズ順に並べて連番を振っているので、元の単語帳とは全く違う連番です。
# CSVデータ作成ガイドライン
## 📝 CSVデータ作成ガイド

このガイドラインに従って、アプリに読み込ませるCSVデータを作成してください。

### 1. 基本ルール

* **1行 = 1つの例文**
* **区切り文字**:
    * **メインの区切り**: **パイプ（`|`）** を使用
    * **チャンク内の区切り**: **ハット（`^`）** を使用
* **チャンクはいくら増えても構わない、必ず適切な読みやすさの長さでチャンクを区切って、全文を用意すること**
* **文脈や一般常識として、英文に誤字や違う単語の間違いがありそうなところは修正してください**

### 2. 書式フォーマット

| ヘッダー | 説明 |
| :--- | :--- |
| **ID** | **通し番号**（例文のファイル名に対応） |
| **英文** | 例文全体 |
| **日本語訳** | 例文の日本語訳 |
| **チャンク1** | 例文を分けた塊（構造分析1） |
| **チャンク2** | 例文を分けた塊（構造分析2） |
| **チャンク3...** | 例文を分けた塊（必要に応じて） |

$$\text{ID} \ | \ \text{英文} \ | \ \text{日本語訳} \ | \ \text{チャンク1} \ | \ \text{チャンク2} \ | \ \text{チャンク3} \ \dots$$

---

### 3. チャンク（塊）の書き方

各**チャンク**は、以下の**4つの情報**を**ハット（`^`）**で区切って記述します。

$$\text{英語の塊} \ \wedge \ \text{文法ラベル} \ \wedge \ \text{タイプ} \ \wedge \ \text{塊の訳}$$

| チャンク内の情報 | 詳細 | 例 |
| :--- | :--- | :--- |
| **英語の塊** | 例文を区切った英語のフレーズ | `I usually go` |
| **文法ラベル** | その塊の文法的な役割 | `Subject + Verb (S+V)` |
| **タイプ** | アプリで色分けするための分類（以下の3つから選択） | `core`、`modifier`、`connector` |
| **塊の訳** | その塊に対応する日本語訳 | `私はたいてい行きます` |

> 📌 **タイプによる色分け**:
> * **core**: 文の骨格（主語・動詞・目的語など）→ **赤色**
> * **modifier**: 修飾語句（前置詞句・不定詞など）→ **緑色**
> * **connector**: 接続詞・関係詞など → **青色**

---

### 4. 記入例とファイル名（ID）の対応

もし音声ファイルが**1.mp3**だった場合、CSVのIDは「**1**」になります。

| ID | 英文 | 日本語訳 | チャンク1 | チャンク2 |
| :---: | :--- | :--- | :--- | :--- |
| **1** | I am studying English. | 私は英語を勉強しています。 | I am studying^Main Verb^core^私は勉強している | English.^Object^core^英語を |

---
